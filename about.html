About the Game

This game was Project 3 in the Udacity Front-End Nanodegree program. We had to create a Frogger-like game using provided visual assets and a game loop engine. The objectives were to better understand JavaScript's object-oriented programming features as well as working with canvas in HTML5.

About the Developer

Sia Karamalegos is a relatively new web developer. She learned Fortran 90 (!!) in college, and then didn't pick up programming again until the advent of MOOCs. After casually playing with Python and JavaScript, she decided to sign up for Udacity's Front End Nanodegree as well as Tech Talent South's code-immersion bootcamp in New Orleans.

Sia loves data and back-end web development but also likes to be able to implement well-designed front-end interfaces. You can learn more about her on her portfolio page.
